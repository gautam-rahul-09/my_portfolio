## üì¶ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/gautam-rahul-09/portfolio-website.git
   cd portfolio-website
   ```

2. **Open in browser**:
   - Simply open `index.html` in your web browser
   - Or use a local server like Live Server in VS Code

3. **Configure EmailJS** (for contact form):
   - Follow the detailed setup guide in `SETUP.md`
   - Update credentials in `emailjs-config.js`

## ‚öôÔ∏è Configuration

### EmailJS Setup
1. Create account at [EmailJS](https://www.emailjs.com/)
2. Set up email service and template
3. Update `emailjs-config.js` with your credentials:

```javascript
const EMAILJS_CONFIG = {
    PUBLIC_KEY: "your_public_key",
    SERVICE_ID: "your_service_id", 
    TEMPLATE_ID: "your_template_id"
};
```

### Customization
- **Colors**: Modify CSS custom properties in `styles.css`
- **Content**: Update personal information in `index.html`
- **Images**: Replace placeholder images with actual project screenshots
- **Links**: Update social media and project links
